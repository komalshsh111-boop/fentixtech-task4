# Anabi Instagram Profile Page

## Project Overview
This project is a mobile-responsive Instagram profile page clone built using only HTML and CSS.

## Features
- Dark theme (black background)
- Mobile responsive design
- Profile picture with stats
- Bio section
- Action buttons (Following, Message)
- Story Highlights
- Navigation tabs with SVG icons
- Post grid layout (3 columns)

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid)
- SVG Icons

## Pages
- index.html — Main profile page
- style.css — External stylesheet

## How to Run
1. Download both files
2. Keep them in same folder
3. Open index.html in any browser
